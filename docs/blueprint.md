# **App Name**: Dynamic Page Editor

## Core Features:

- Edit Mode Toggle: Toggle between user and edit mode with OTP authentication.
- Configurable Buttons: Add custom buttons with configurable text, color, icons, and URLs.
- Text Elements: Add configurable text elements anywhere on the page with custom font size and type.
- Icon Integration: Add and position icons from a library anywhere on the page.
- Drag and Drop: Drag and drop functionality for re-positioning buttons and text elements in edit mode.
- REST Call Execution: Execute REST calls associated with each button click, displaying a loading animation and status indicator. The base of the URL is configurable.
- Configuration Export: Display a JSON representation of the page configuration that can be copied and pasted into the configuration area of the HTML file.

## Style Guidelines:

- Primary color: Medium sky blue (#87CEEB) for a clean, modern feel.
- Background color: Light gray (#F0F0F0) to provide a neutral backdrop.
- Accent color: Soft purple (#A099DE) for interactive elements and highlights.
- Font pairing: 'Poppins' for headlines and 'PT Sans' for body text.
- Use Font Awesome icons for visual cues and button enhancements.
- Grid-based layout to facilitate easy drag and drop of elements.
- Subtle spinning animation for REST call progress, green check for success, red X for error.
